About GameNest

GameNest is your one-stop destination to access several different simple browser games. It is an online hub hosting a wide variety of games with a uniform, aesthetically-pleasing GUI.




Technology Overview

Tech Stack: MERN. React client, Express and Node.js server, and MongoDB Database.

Language(s): JavaScript

Library(s): React

Hosting: Azure

Wordle

This will be a simple Wordle game in which the user has 6 attempts to guess a 5-letter word. As the user guesses words, their letters will be highlighted in gray if the letter is not present in the word, yellow if the letter is in the word but is in the wrong spot, and green if the letter is in the word and is already in the right spot.

Wordle will utilize a database to keep track of the user's scores (assuming they are logged in). Additionally, API calls to a separate web server will supply new random words everytime the user begins a new game.

In the future, I may implement a feature where users can choose to guess words longer than the default 5 letters.
